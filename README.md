# Waxis AI Control Plane

Status: product requirements repository

Primary document:

- [PRD.md](./PRD.md)

This repository defines the Waxis Enterprise AI Control Plane solution: a governed model gateway, policy layer, tool connector registry, observability system, and evaluation workflow for production AI applications.

The PRD is based on Waxis website positioning plus a June 2026 research snapshot across official AI platform, safety, observability, and enterprise-agent sources.
